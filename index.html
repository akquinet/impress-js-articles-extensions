CoffeeScript

What is CoffeeScript

  • Javascript without the fail
  • cleaner syntax
  • far less bloat
  • compiles to Javascript

Wait! What? I though Javascript is the problem!

Javascript has good parts

Source: Anongallery
  • Coffee sticks to the good part
  • Coffee generates readable, pretty-printed Javascript
  • Coffee generats JSLint compliant Javascript
  • Coffee gets rid of many WTFs
  • Interoperable with nearly all existing JS libraries (jQuery, Node, etc...)

Coffee Hello World

// Generated Javascript:
<script type="text/coffeescript">
    if 1 == '1' then alert 'WTF?!' else alert ':-)'
</script>

One caveat...

significant whitespace instead of braces

Coffee Methods

// Generated Javascript:
<script type="text/coffeescript">
    hello = (name) ->
        greeting = "Hello #{name}"
    alert hello 'World'
</script>

Coffee Object Model

// Generated Javascript:
<script type="text/coffeescript">
    class Developer
        constructor: (@name, @favoriteBrewStyle) ->

        brew: ->
            alert "Brewing #{@favoriteBrewStyle}"
    mo = new Developer "Moritz", "Espresso"
</script>

Coffee Goodies - while/until returns an array

// Generated Javascript:
<script type="text/coffeescript">
    count = 0
    addicted = until count > 5
        count++
        "Drinking Coffee #{count}"
</script>

["Drinking Coffee 1", "Drinking Coffee 2", "Drinking Coffee 3", "Drinking Coffee 4", "Drinking Coffee 5", "Drinking Coffee 6"]

Coffee Goodies - the ?

// Generated Javascript:
<script type="text/coffeescript">
    if foo?
        alert "Foo: #{foo.toUpperCase()}"
</script>

Coffee Goodies - List comprehension

// Generated Javascript:
<script type="text/coffeescript">
    list = [1 .. 10]
    evenDoubled = (item * 2 for item in list when (item % 2 == 0))
    alert(evenDoubled)
</script>
