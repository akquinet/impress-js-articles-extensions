Jira Issue Updater Plugin

  • offers to execute a transition according to the job result
  • transition executions can be commented automatically
  • requires jira filter
  • (-) plenty of configuration steps compared to a lot other plugins
  • (-) occasionally dangerous, if jira filter configuration inaccurate
